# Changelog

## [1.0.0] - 2026-04-19
### Changed
- Rebranded and refactored from "Antigravity Sync" to "Antigravity Auto Retry".
- Completely removed Git sync functionality, side panel, and related dependencies to focus solely on the CDP-based auto-retry automation.
- Improved macOS CDP patching command to be more robust and dynamic.
- Switched from Yarn to npm as the preferred package manager.
- Simplified logging to use VS Code Output Channel instead of console.
- Removed Webview UI and status bar indicator for a more lightweight, background-focused extension.

## [0.3.3] - Previous versions
- Legacy versions containing the Git sync feature.
